Insaat ERP — Masaustu (Windows) adim adim

Bu dosyayi cift tiklayin: Internet Explorer degil, Chrome / Edge acsin. .txt acilmazsa yine bu HTML kullanin.

Ne bu?

Masaustu: "Insaat ERP" penceresi. API: ayni PC veya sunucuda 3000 portu (ornek). Uygulama veriyi API'den alir; API kapaliyken giris/ liste calismayabilir.

Ne kopyalamaniz gerekir?

Proje aracinda insaat-erp-backend ve insaat-erp-frontend klasorlerinin tamamini (ornek: C suresinde bir klasor icine).

Bolum A — Ilk kurulum

0) Node.js LTS: https://nodejs.org — LTS indirin, kurun.

1) PowerShell acin, frontend klasorune girin (yolu kendi yolunuzla degistirin):

cd C:\SIZIN_KLASOR\insaat-erp-frontend

2) Paketler (internet):

npm install

3) (Istege bagli) Hizli pencere + API ayni PC'de. Once API'yi baska pencerede: insaat-erp-backend icinde npm start (veya npm run dev). Sonra:

cd C:\SIZIN_KLASOR\insaat-erp-frontend npm run electron:dev

4) Kurulabilir .exe icin (release klasorune yazar):

cd C:\SIZIN_KLASOR\insaat-erp-frontend npm run build:desktop npm run pack:win

5) Cikti: insaat-erp-frontend\release\ — icinde kurulum .exe veya portable .exe

6) Masaustu acilinca API ciksin: varsayilan http://127.0.0.1:3000 (frontend icinde .env.electron).

Bolum B — Guncelleme

Yeni dosyalari uzerine kopyalayin, sonra ayni frontend klasorunde:

npm install npm run build:desktop npm run pack:win

API baska bilgisayardaysa

insaat-erp-frontend\.env.electron icinde VITE_API_BASE= satirini o sunucunun adresiyle guncelleyin; sonra yine build:desktop ve pack:win calistirin.

Sorun: Endpoint bulunamadi / bos ekran

API calismiyor veya yanlis adreste. Backend klasorunda test:

cd C:\SIZIN_KLASOR\insaat-erp-backend node scripts\verify-api.cjs

200 ve "surum" = 2 benzeri gorunmuyorsa, yanlis proje calisiyor veya port karisik.
